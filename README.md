# ⛪ Relatório de Contagem - PIB Floripa

Este aplicativo foi desenvolvido para facilitar a coleta e organização dos dados de frequência dos cultos de domingo da **Primeira Igreja Batista de Florianópolis**.

## 🚀 Funcionalidades
* **Sincronização de Data:** A data preenchida no primeiro culto é automaticamente replicada para o segundo.
* **Cálculos Automáticos:** Soma total de membros, visitantes, ministério infantil (MI), pré-adolescentes (MPA), Ebed e Batismo.
* **Relatório Formatado:** Gera um resumo textual pronto para ser copiado.
* **Integração com WhatsApp:** Botão direto para enviar o relatório final aos responsáveis.
* **Design Responsivo:** Otimizado para uso em smartphones com fontes grandes e campos de fácil leitura.

## 📱 Como instalar no Celular (PWA)
Para usar como um aplicativo sem precisar baixar da loja:

### No Android (Chrome):
1. Acesse o link do App.
2. Toque nos **três pontinhos** (canto superior direito).
3. Selecione **"Instalar aplicativo"** ou **"Adicionar à tela inicial"**.

### No iPhone (Safari):
1. Acesse o link do App.
2. Toque no botão de **Compartilhar** (quadrado com seta para cima).
3. Selecione **"Adicionar à Tela de Início"**.

## 🛠️ Tecnologias Utilizadas
* [Streamlit](https://streamlit.io/) - Framework para a interface web.
* [Python](https://www.python.org/) - Lógica de programação e cálculos.
* [GitHub](https://github.com/) - Hospedagem do código fonte.

## 🔐 Segurança
O acesso ao formulário é protegido por senha para garantir que apenas voluntários autorizados enviem os dados.
